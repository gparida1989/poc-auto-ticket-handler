# App Test Guide

## How to Start the API Server

### 1. Prerequisites
- Python 3.8+
- All dependencies installed (see requirements.txt)
- Project structure:
  - src/
    - main.py
    - ...

### 2. Activate Virtual Environment (if used)
On Windows:
```
venv\Scripts\activate
```
On macOS/Linux:
```
source venv/bin/activate
```

### 3. Start the FastAPI Server
From the project root (`poc-auto-ticket-handler`):
```
uvicorn src.main:app --reload
```
- `--reload` enables auto-reload on code changes (for development)
- Default address: http://127.0.0.1:8000/

### 4. Test the API
- Open your browser or Thunder Client/Postman
- Visit: http://127.0.0.1:8000/
- You should see a JSON response or FastAPI welcome page

#### API Docs (Swagger UI)
- Visit: http://127.0.0.1:8000/docs
- Interactive API documentation (auto-generated by FastAPI)

#### Health Check
- GET http://127.0.0.1:8000/health
- Should return `{ "status": "ok" }`

### 5. Troubleshooting
- If you see "connection refused":
  - Make sure the server is running and not blocked by firewall
  - Check terminal for errors
  - Confirm you are using the correct port (default: 8000)
- If you change code, the server will auto-reload (if using `--reload`)
